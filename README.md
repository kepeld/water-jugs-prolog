# Задача переливання (Water Jugs Problem)

**Автор:** Козін Андрій  
**Курс:** Логічне програмування, НаУКМА

## Опис

Класична задача переливання води між посудинами. Використано BFS для оптимального розв'язку.

## Файли

- `water_jugs.pl` — реалізація на Prolog
- `WaterJugs.hs` — реалізація на Haskell (для порівняння)
- `Kozin_Final.pdf` — презентація

## Запуск

**Prolog:**
```prolog
?- [water_jugs].
?- demo.
```

**Haskell:**
```bash
runhaskell WaterJugs.hs
```
